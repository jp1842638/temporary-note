# Temporary Note

A minimal browser-based scratchpad for jotting things down for a moment.

**Nothing is saved.** No cookies, no local storage, no server. Refresh the page and your note is gone — that's the whole point.

## Features

- Clean, distraction-free editor
- Rich-text basics with familiar shortcuts
- Live character count
- One-click clear button
- Pure HTML / CSS / JS — no build step, no dependencies

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl` + `C` | Copy |
| `Ctrl` + `V` | Paste |
| `Ctrl` + `X` | Cut |
| `Ctrl` + `A` | Select all |
| `Ctrl` + `B` | Bold |
| `Ctrl` + `I` | Italic |
| `Ctrl` + `U` | Underline |
| `Ctrl` + `Shift` + `V` | Paste as plain text |
| `Ctrl` + `Z` | Undo |
| `Ctrl` + `Y` | Redo |

> On macOS, use `Cmd` instead of `Ctrl`.

## Usage

Just open `index.html` in any modern browser. That's it.

```bash
open index.html
```

Or host the folder as a static site (GitHub Pages, Netlify, Vercel, etc.).
